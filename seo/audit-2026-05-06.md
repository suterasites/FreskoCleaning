## Phase plan

### Phase 3a - Mechanical (this session)
- Update `<head>`: tighten title, drop keywords meta, add canonical, expand OG, add Twitter, add geo, swap to `@graph` schema with LocalBusiness + WebSite + 4x Service nodes + BreadcrumbList.
- Add skip-to-content link as first body child.
- Wrap content in `<main id="main">`.
- Add explicit `width`/`height` attributes to all content `<img>` tags (hero, service tiles, gallery, about, logo).

### Phase 3b - Site infrastructure (this session)
- `sitemap.xml` with single URL entry, lastmod 2026-05-06.
- `robots.txt` allowing all + Sitemap line.

### Phase 3c - Deploy (this session)
- Commit + push to GitHub. Cloudflare Pages auto-deploys (single page, fast).
- Submit sitemap in Google Search Console after propagation.
- Re-run PageSpeed mobile/desktop to capture CLS gains from img dimensions.

### Phase 2 - Voice references (backlog)
Source: existing site copy is plain and direct (Luke wrote captions personally). Facebook captions + 5-question email to Luke would seed `voice.md`, `stats.md`, `opinions.md`, `stories.md`. Skip until subscription or follow-up build is on the table - this is a one-off.

### Phase 4 - Content expansion (backlog, conditional)
Single-page format limits Phase 4. Options if Luke wants more SEO juice later:
- Convert to multi-page (4 service pages + gallery + about + contact).
- Add FAQ block to homepage covering Wagga-specific search ("how often should I clean my gutters in Wagga", "best time of year to pressure wash NSW", "post-build clean checklist").
- Expand "Where We Work" to enumerate Turvey Park, Mount Austin, Kooringal, Estella, Lake Albert, Bourkelands, Tatton, Forest Hill (priming for Phase 5 city LPs).

### Phase 5 - City landing pages (backlog)
Wagga has limited surrounding population centres. If Luke wants regional reach later, mirror Grout Academy `/regrouting-geelong/` template for: Junee, Coolamon, The Rock, Henty, Tumbarumba, Cootamundra, Gundagai, Albury (border crossover risk - check CPC).
